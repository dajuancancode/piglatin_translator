Hey there! My name is DaJuan Harris, I am a student at Broward College studying for an AA so I can transfer into a computer science program. This translator was built with django, HTML, and CSS. This is not my first website, nor is this my first forray into backend development. This is however, my first time using django, so that is something to celebrate.

If you're interested in how this translator works, I'll tell you. So on the homepage, there is a form with two inputs; one text field and one submit button. The form has an action that points to the translate page, so when you click submit, that is where you will be taken. The text field has a name of translation so that I can work with it in the translate view. In the translate view, I take the translation and split the sentence into an array. I then iterate over every word in the loop and perform the act of translating it.
(If you want to know how pig latin is translated, click this wikipedia link.) After translating each word in the array, I join the words back into a sentence. Then I export both the translation and the original text to the translate page for display.
